# Normalización con Pandas

Objetivo: Optimizar la estructura de datos y mejorar su eficiencia y capacidad de consulta a travez de un esquema copo de nieve. 

Al aplicar este enfoque, se busca reducir la redundancia y minimizar la duplicación de datos en la base de datos, lo que conlleva un almacenamiento más efectivo y consultas más rápidas.

Como resultado de esta normalización se obtuvo una base de dato más flexible y escalable, adecuada para entornos analíticos y de generación de informes.

Este modelo desarrollado se termino subiendo a una base de datos PostgreSQL, alojado en la nube por medio de la Plataforma cloud aiven.io
